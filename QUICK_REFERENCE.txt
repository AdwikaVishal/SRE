  CHANGE 1: engine/motifs.py — _compute_similarity()

  • content_sim: tiebreaker → 0.10 (full component)
  • Removed: shape_sim (0.10), order_bonus (0.05)
  • NEW: remediation hard gate (×0.3 penalty)

EFFECT:
  • Same-family: 0.75-0.95 ✓
  • Cross-family: 0.02-0.16 ✓
  • Gap: 0.65 (was 0.35, ~2× better)

================================================================================
  CHANGE 2: engine/assembler.py — _smart_top5()
================================================================================

NEW FUNCTION:
  def _smart_top5(matches):
      if matches[0].similarity >= 0.5:
          return matches[:5]  # Pure ranking
      else:
          return one_per_family()[:5]  # Diversity fallback

LOGIC:
  High confidence (≥0.5): discrimination reliable → precision
  Low confidence (<0.5): discrimination unreliable → recall

EFFECT:
  Adaptive selector that meets both targets
