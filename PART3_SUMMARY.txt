                          OPTIMIZATION MODES
================================================================================

FULL PIPELINE:
  python run_optimization.py --mode full --output results/
  Steps: 1) Baseline diagnostic run
         2) Weight sweep on key parameters
         3) Failure analysis
         4) Targeted optimization for weak metrics
         5) Final validation with best config
  Time: ~60-90 minutes (3 seeds, fast mode)

WEIGHT SWEEP:
  python run_optimization.py \
    --mode sweep \
    --param stageA_min_similarity 0.50 0.55 0.60 0.65 0.70 \
    --param decoy_similarity_cap 0.35 0.40 0.45 0.50 \
    --output results/sweep/
  Time: ~30-60 minutes (50 configs, 3 seeds)

ABLATION STUDY:
  python run_optimization.py \
    --mode ablation \
    --baseline baseline.json \
    --variants variant1.json variant2.json \
    --output results/ablation/
  Time: ~20-30 minutes (5 configs, 3 seeds)

TARGETED OPTIMIZATION:
  python run_optimization.py \
    --mode targeted \
    --metric precision@5_mean \
    --initial config.json \
    --output results/precision_opt/
  Time: ~10-15 minutes (3 recommendations, 3 seeds)

VALIDATION:
  python run_optimization.py \
    --mode validate \
    --config best_config.json \
    --seeds 9999 31415 27182 \
    --output results/validation/
  Time: ~10-15 minutes (3 seeds)
