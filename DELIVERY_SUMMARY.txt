PHASE 2: SWEEP (1-2 hours)
  Command: python run_optimization.py --mode sweep --param ... --seeds ...
  Output:  sweep_results.json (configs ranked by score)
  Action:  Find best parameters from grid search
  Result:  Typically +1-2% improvement

PHASE 3: TARGETED (1-2 hours)
  Command: python run_optimization.py --mode targeted --metric ... --initial ...
  Output:  targeted_optimization.json
  Action:  Focus on weakest remaining metric
  Result:  Typically +2% more improvement

PHASE 4: ABLATION (1 hour)
  Command: python run_optimization.py --mode ablation --baseline ... --variants ...
  Output:  ablation_report.json (metric deltas)
  Action:  Compare configs, verify no regressions
  Result:  Confidence in final choice

PHASE 5: VALIDATE (30 minutes)
  Command: python run_optimization.py --mode validate --config ... --seeds ...
  Output:  validation_report.json (final scores on all seeds)
  Action:  Confirm improvement holds
  Result:  Score advances toward 60%
